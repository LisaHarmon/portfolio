# Portfolio Sample: Software User Guide
---

## About This Sample

### Disclaimer

This user guide describes *Smart Shopper*, a fictitious software application created for portfolio and demonstration purposes. The application, its features, and all associated documentation are entirely hypothetical and do not represent an actual product or service. 

Any resemblance to existing applications, products, services, or companies is unintentional and purely coincidental. All product names, features, and workflows described in this document are intended solely to demonstrate my technical writing skills and practices.

### Problem

Currently, consumers who use grocery shopping apps must use multiple sites to complete shopping tasks: creating lists in one app, comparing prices in another, and locating store products in yet another. Users need an easier, streamlined way to make lists, find the best prices, and locate items in whatever store they choose. 

### Solution

To help shoppers save time and money, I did the following:

* Developed a (fictional) mobile app that centralizes grocery shopping app features 
* Designed user-facing content that guides the app users through setup, navigation, and task completion
