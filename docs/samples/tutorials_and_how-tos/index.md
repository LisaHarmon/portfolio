# Portfolio Sample: How-to Guide
---

## About This Sample

This guide is a redacted version of a document I created for a Coast Guard program I supported while working as a government contractor at JPI. The guide explains how to log action items in a custom SharePoint list, enabling team members to capture key task information and transition smoothly from one topic to the next during program meetings.

### Tools Used

This guide was originally developed in Microsoft Word and stored in a SharePoint library. To migrate the content and publish it on this site, I used:

* Visual Studio Code
* Markdown
* GitHub
* Jekyll
