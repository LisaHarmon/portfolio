# Creating a WIN10 Program Action Item
---

## Introduction

These instructions describe how to create an action item raised at a WIN10 program meeting.

{: .important }
Do not use these instructions to log risks, issues, or decisions. If you are logging one of these items, refer to the **Win10 RAID Log guide** [*resource redacted*].

## Before You Begin

Ensure you can access the **CG Windows 10 Migration Project SharePoint** site [*link redacted*]. To request access, contact [*contact name and link redacted*].

## Steps

1. Open the **CG Windows 10 Migration Project** site [*link redacted*]. 

2. Go to the **Lists** menu, and select **WIN10 Program Action Items**.

    [*image redacted*]

    The list appears.

    [*image redacted*]

3. Go to the menu bar, and select **List Tools** > **Items**.

    [*image redacted*]

4. On the ribbon, select **New Item** > **Item**.

    [*image redacted*]

    The WIN10 Program Action Items – New Item form appears.

    [*image redacted*]

5. Complete the form fields as follows:

    | Field            | Action           | Required?        |
    |:-----------------|:-----------------|:-----------------|
    | Title            | Enter a brief description conveying the item's main theme. | Yes | 
    | Description      | Enter additional details about the item. | Yes |
    | Artifact         | Add a link to the document associated with the item. | No | 
    | Meeting Date     | Select the date on which the item was assigned. | Yes |
    | Due Date         | Select the date by which the item should be completed. | Yes |
    | Status           | Select the item's current state: New, In progress, Delayed, Overdue, On hold, or Completed. | Yes |
    | Priority         | Select the item's current ranking in relation to other action items: Low, Medium, or High. | Yes |
    | Action Org       | Enter the responsible organization. | Yes |
    | Action Officer   | Enter the name of the person responsible for completing the item. | Yes |
    | Comments         | Enter any additional notes about the item. | No |
    | Dependency       | Enter any other tasks that must be completed before the item can be resolved. | No |


6. Click **Save**.

    Your action item is added to the list.
